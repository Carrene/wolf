## Delete a token

### DELETE /apiv1/tokens/:token_id

Delete a token by id

### Url Parameters

Name | Example
--- | ---
token_id | 1

### Response: 200 OK

#### Headers

* Content-Type: application/json; charset=utf-8
* Content-Length: 340

#### Body

```json
{
    "id":1,
    "phone":989121234567,
    "expireDate":"2000-12-07",
    "createdAt":"2018-04-24T23:29:40.093087Z",
    "name":"first_token",
    "isExpired":true,
    "modifiedAt":null,
    "cryptomodule":{
        "oneTimePasswordLength":4,
        "id":1,
        "timeInterval":60
    },
    "isActive":true,
    "provisioning":null
}
```
